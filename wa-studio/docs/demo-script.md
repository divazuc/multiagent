# תסריט דמו ללקוח — סוכן WhatsApp חכם

> משך מומלץ: 10–12 דקות. שני מסכים: **דשבורד הלקוח** (`/demo.html`) ו-**WA Studio** (המסך הראשי).

---

## הכנה לפני הפגישה (חובה!)

1. **Supabase ער** — היכנס לדשבורד של Supabase או שלח הודעה אחת דרך ה-Studio ביום הפגישה. פרויקט חינמי נרדם אחרי שבוע ללא פעילות — אם זה קורה באמצע פגישה אין דרך להתאושש בזמן אמת. *שקול שדרוג ל-Pro לפני פגישות חשובות.*
2. הפעל את שני השרתים:
   - `cd server && npm run dev` (סוכן, פורט 8080)
   - `cd wa-studio && npm run dev` (ממשק, פורט 5174)
3. ודא שבכותרת ה-Studio מופיע **Agent online** בירוק.
4. ודא שרשימת הסשנים נקייה (סשני QA מוסתרים כברירת מחדל — אל תלחץ על כפתור ה-QA בפגישה).
5. פתח שני טאבים מראש:
   - `http://localhost:5174/demo.html` — דשבורד הלקוח
   - `http://localhost:5174` — ה-Studio
6. גיבוי: צלם מסכים מראש למקרה של תקלת אינטרנט.

## מבנה הדמו — סיפור בארבע מערכות

### מערכה 1 — הפתיח: "ככה זה נראה מהצד שלך" (2 דק׳)

פתח עם **דשבורד הלקוח** (`/demo.html`). זה המסך הכי קל לעיכול — בהיר, בעברית, בלי מונחים טכניים.

מה להראות ומה להגיד:
- **תיבת הלידים**: "כל מי שכותב לוואטסאפ של העסק הופך אוטומטית לליד עם היסטוריה מלאה."
- **🤖 סיכום הסוכן** (האלמנט החזק במסך): "אתה לא צריך לקרוא את כל השיחה — הסוכן מסכם לך כל לקוח במשפט-שניים."
- **"מה הסוכן כבר יודע על הלקוח"**: "תוך כדי שיחה הסוכן מזהה צורך, תקציב ודחיפות — זה לא צ'אטבוט, זה איש מכירות שמאבחן."
- **השיחה האחרונה**: גלול בעדינות — בועות וואטסאפ מוכרות, עברית טבעית עם אימוג'י.

> 💡 טיפ: אם הלקוח שואל "זה אמיתי?" — זה בדיוק הפתח למערכה 2.

### מערכה 2 — ההוכחה: שיחה חיה (3 דק׳)

עבור ל-**WA Studio**, בחר את הסשן החי (Leadz marketing).

1. הצג את עצמך כלקוח: כתוב הודעה כמו לקוח אמיתי, למשל: *"היי, כמה עולה לבנות אתר לעסק קטן?"*
2. בזמן שמחוון ההקלדה רץ (10–15 שניות) — **דבר, אל תחכה בשקט**: "הסוכן עכשיו קורא את פרופיל העסק, את ה-FAQ ואת היסטוריית השיחה, ומנסח תשובה בטון שהוגדר לעסק."
3. כשהתשובה מגיעה — הצבע על: עברית טבעית, אימוג'י מדוד, ושאלת המשך שמקדמת מכירה (לא רק עונה).
4. **פאנל DB State מימין**: "שים לב — תוך כדי השיחה הוא כבר רשם שהלקוח צריך אתר לעסק קטן. זה נשמר אוטומטית לליד."

### מערכה 3 — ההקמה: "ככה מגדירים אותך" (3 דק׳)

הדגם כמה מהר מקימים סוכן לעסק חדש:

1. **+ New Business** → הקלד את **שם העסק של הלקוח שמולך** (פרסונליזציה = וואו).
2. Mode: Setup → **Start Session**.
3. הסוכן פותח בעברית ושואל על סוג העסק — ענה תשובה אחת-שתיים בשם הלקוח.
4. הסבר את ההמשך במילים (אל תעבור את כל האשף בלייב): "הוא ימשיך לשאול על שירותים, קהל יעד, סגנון דיבור — ובסוף בונה לבד את פרופיל העסק ושותל שאלות נפוצות מותאמות לתחום."
5. אזכר את **שלב הדמו/למידה**: "אחרי ההקמה הסוכן לומד את סגנון הדיבור שלך — איך אתה עונה ללקוחות — ומחקה אותו."

### מערכה 4 — השליטה: שקיפות מלאה (2 דק׳)

1. **FAQ / Knowledge Base**: "הסוכן גם מציע לבד שאלות שכדאי להוסיף על סמך מה שלקוחות באמת שואלים" — הצבע על badge ההצעות ואשר אחת בלחיצה.
2. **Runs** (אם הלקוח טכני): "כל תשובה מפורקת לצעדים עם זמנים — אין קופסה שחורה."
3. **Admin**: "מבט-על על כל העסקים — בריאות, הודעות, עלויות Meta משוערות. רוב העסקים נשארים בטיר החינמי של 1,000 שיחות בחודש."

### סגירה

חזור לדשבורד הלקוח: "בסוף — זה מה שאתה רואה כל בוקר: לידים מסוכמים, מתויגים, עם כל ההיסטוריה. הסוכן עובד 24/7 ואתה מתערב רק כשצריך."

## רגעי "וואו" לחדד

| רגע | למה זה עובד |
|---|---|
| סיכום AI על כל ליד | חוסך ללקוח את הפעולה שהוא הכי שונא — לקרוא שיחות ארוכות |
| צ'יפים של אבחון (צורך/תקציב) שנצברים תוך כדי שיחה | ממחיש שהסוכן *מוכר*, לא רק עונה |
| הקמת עסק בשם של הלקוח בלייב | פרסונליזציה מיידית — "זה יכול להיות שלי מחר" |
| התשובה החיה בעברית עם המשך שיווקי | ההוכחה שאין טריק — זה עובד עכשיו |

## מוקשים להימנע מהם

- **אל תשלח הודעות ג'יבריש** — הסוכן יענה ברצינות וזה מבזבז את הרגע.
- **אל תפתח את מתג ה-QA** — 27 סשני בדיקה לא מוסיפים כבוד.
- **אל תעמוד בשקט בזמן ההקלדה** (10–15 שניות) — זה החלון לספר מה קורה מאחורי הקלעים.
- **אל תדגים את כל אשף ההקמה** — שני צעדים ואז לספר במילים. אשף מלא לוקח 5–7 דקות ושוחק את הקצב.
- אם משהו נשבר — עבור לצילומי המסך שהכנת ותמשיך בסיפור.

- אם אין לידים אמיתיים לעסק — הדף מציג נתוני הדגמה איכותיים (מסומנים "נתוני הדגמה").
- ניקוי סשן לפני פגישה: Restart מהדיאלוג בבחירת עסק עם סשן קיים.
